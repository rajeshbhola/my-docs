## Sample Advanced Programming Questions

### Thread-Safe Programming
```java
// Implement custom thread-safe Singleton pattern
class Singleton {
    private static volatile Singleton instance;
    private Singleton() {}
    
    public static Singleton getInstance() {
        if (instance == null) {
            synchronized (Singleton.class) {
                if (instance == null) {
                    instance = new Singleton();
                }
            }
        }
        return instance;
    }
}
```
